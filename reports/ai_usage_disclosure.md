# AI Usage Disclosure

## AI Tools Used

The following AI tool was used during this assignment:

* ChatGPT

## AI-Assisted Sections

AI assistance was used for:

* Understanding the assignment requirements
* Planning the project scope
* Designing the project folder structure
* Creating the pipeline structure
* Drafting and improving Python script structure
* Debugging runtime errors
* Explaining data engineering decisions
* Drafting README and documentation content
* Structuring the final report outline

## Human Review and Validation

All AI-assisted outputs were reviewed before being included in the project.

Validation steps included:

* Running each Python script locally
* Running the full pipeline using `python src/run_pipeline.py`
* Checking that expected CSV outputs were generated
* Checking that EDA figures were saved successfully
* Reviewing terminal logs for successful execution
* Reviewing generated outputs before committing them to GitHub

## Modifications Made

AI-generated suggestions were modified to match:

* The selected city: Melbourne
* The local project folder structure
* Actual downloaded Inside Airbnb files
* Errors encountered during execution
* Assessment deliverable requirements

## Critical Assessment of AI Outputs

Some AI suggestions required correction during implementation.

Examples include:

* SQL numeric fields needed explicit casting in DuckDB using `TRY_CAST`
* Statistical tests needed safeguards for insufficient sample sizes
* Git ignore rules needed adjustment so report outputs were included for review
* Code was adjusted to use professional logging instead of informal progress messages

## Responsibility Statement

AI was used as a support tool, not as a replacement for project ownership.

The final implementation, execution, validation, and submission decisions remain the responsibility of the candidate.
